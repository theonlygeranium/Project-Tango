# Project Tango

Project Tango is an AI-powered web application designed to provide users with ready-to-use, specialized voice agents for diverse tasks such as therapy, general conversation, and information retrieval. Built on a modern, open-source voice framework, the platform prioritizes ultra-low latency, real-time transcriptions, and highly customizable agent behaviors.

[![Deploy](https://github.com/theonlygeranium/Project-Tango/actions/workflows/deploy.yml/badge.svg)](https://github.com/theonlygeranium/Project-Tango/actions/workflows/deploy.yml)

**Live URL:** [https://project-tango.schubert.life](https://project-tango.schubert.life)

## Tech Stack
- **Frontend:** Next.js / React 15.x
- **Voice Framework:** Pipecat (by Daily) 0.x
- **Transport:** WebRTC (via LiveKit or Daily)
- **STT:** Deepgram Nova-3
- **TTS:** ElevenLabs Flash v2.5
- **LLM Gateway:** LiteLLM 1.x
- **Local LLM:** Qwen 3.5 (via Ollama)
- **Cloud LLM:** WRITER Palmyra
